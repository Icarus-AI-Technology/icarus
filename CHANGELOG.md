# Changelog

All notable changes to ICARUS v5.0 will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [5.0.3] - 2025-11-15

### Added
- Initial project setup with Vite + React + TypeScript
- Dark Glass Medical Design System with neumorphic components
- Supabase PostgreSQL integration
- Core UI components (Button, Card, Toast)
- Layout components (Sidebar, Header)
- Dashboard page with stats and IA insights
- IcarusBrain AI service structure
- PWA configuration with offline support
- Comprehensive documentation (CLAUDE.md, .clinerules)
- GitHub Actions CI/CD workflow
- Testing infrastructure (Jest + Playwright)

### Infrastructure
- Vite 6.0 build system
- Tailwind CSS 4 with custom neumorphic utilities
- React Query for server state management
- Zustand for client state management
- TypeScript 5.6 with strict mode

### Documentation
- README.md with quick start guide
- CLAUDE.md context for AI development
- .clinerules for development standards
- Environment configuration templates
